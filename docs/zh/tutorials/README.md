# 教程

**面向学习的**初学者分步指南。

教程旨在帮助您通过实践学习 KTTC。每个教程都是一个完整的课程，带您从开始到结束。

## 可用教程

1. **[快速入门指南](quickstart.md)** ⭐ *从这里开始！*
   - 安装 KTTC
   - 运行您的第一次质量检查
   - 理解 MQM 评分
   - *时长：5分钟*

2. **[您的第一次批量处理](first-batch.md)**
   - 处理多个翻译
   - 使用 CSV 文件
   - 生成报告
   - *时长：10分钟*

3. **[带自动纠错的翻译](translation-with-qa.md)**
   - 生成 AI 翻译
   - 自动质量检查
   - 迭代优化（TEaR 循环）
   - *时长：10分钟*

## 学习路径

```
快速入门（5分钟）
    ↓
第一次批量处理（10分钟）
    ↓
带自动纠错的翻译（10分钟）
    ↓
准备探索指南！ →
```
