# آموزش‌ها

**یادگیری محور** راهنماهای گام به گام برای مبتدیان.

آموزش‌ها برای کمک به یادگیری KTTC از طریق انجام کار طراحی شده‌اند. هر آموزش یک درس کامل است که شما را از ابتدا تا انتها راهنمایی می‌کند.

## آموزش‌های موجود

1. **[راهنمای شروع سریع](quickstart.md)** ⭐ *از اینجا شروع کنید!*
   - KTTC را نصب کنید
   - اولین بررسی کیفیت خود را اجرا کنید
   - امتیازهای MQM را درک کنید
   - *زمان: 5 دقیقه*

## مسیر یادگیری

```
شروع سریع (5 دقیقه)
    ↓
آماده کاوش در راهنماها! →
```

## بعد از آموزش‌ها

پس از تکمیل این آموزش‌ها، آماده هستید برای:

- **[راهنماها](../guides/README.md)** - راهنماهای چگونه خاص کار
- **[مرجع](../reference/README.md)** - مستندات کامل API
- **[توضیح](../explanation/README.md)** - درک اینکه KTTC چگونه کار می‌کند
