# راهنماها

**وظیفه‌محور** راهنماهای چگونه برای کارهای خاص.

راهنماها به شما کمک می‌کنند تا اهداف خاصی را محقق کنید. هر راهنما فرض می‌کند که شما اصول اولیه را می‌دانید و می‌خواهید مشکل خاصی را حل کنید.
