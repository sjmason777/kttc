# Tutorials

**Learning-oriented** step-by-step guides for beginners.

Tutorials are designed to help you learn KTTC by doing. Each tutorial is a complete lesson that takes you from start to finish.

## Available Tutorials

1. **[Quickstart Guide](quickstart.md)** ⭐ *Start here!*
   - Install KTTC
   - Run your first quality check
   - Understand MQM scores
   - *Time: 5 minutes*

2. **[Your First Batch Processing](first-batch.md)**
   - Process multiple translations
   - Work with CSV files
   - Generate reports
   - *Time: 10 minutes*

3. **[Translation with Auto-Correction](translation-with-qa.md)**
   - Generate AI translations
   - Automatic quality checking
   - Iterative refinement (TEaR loop)
   - *Time: 10 minutes*
