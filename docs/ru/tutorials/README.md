# Учебные руководства

**Учебные** пошаговые руководства для начинающих.

Эти руководства помогут вам изучить KTTC на практике. Каждое руководство представляет собой полноценный урок, который проведет вас от начала до конца.

## Доступные руководства

1. **[Руководство по быстрому старту](quickstart.md)** ⭐ *Начните здесь!*
   - Установка KTTC
   - Запуск первой проверки качества
   - Понимание оценок MQM
   - *Время: 5 минут*

2. **[Ваша первая пакетная обработка](first-batch.md)**
   - Обработка нескольких переводов
   - Работа с CSV-файлами
   - Создание отчетов
   - *Время: 10 минут*

3. **[Перевод с автокоррекцией](translation-with-qa.md)**
   - Создание ИИ-переводов
   - Автоматическая проверка качества
   - Итеративное улучшение (цикл TEaR)
   - *Время: 10 минут*

## Траектория обучения

```
Быстрый старт (5 мин)
    ↓
Первая пакетная обработка (10 мин)
    ↓
Перевод с автокоррекцией (10 мин)
    ↓
Готовы изучать инструкции! →
```
