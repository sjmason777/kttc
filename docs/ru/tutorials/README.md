# Руководства

**Ориентированные на обучение** пошаговые руководства для начинающих.

Руководства предназначены для изучения KTTC через практику. Каждое руководство представляет собой полный урок, который проводит вас от начала до конца.

## Доступные руководства

1. **[Руководство по быстрому старту](quickstart.md)** ⭐ *Начните здесь!*
   - Установите KTTC
   - Запустите первую проверку качества
   - Разберитесь с оценками MQM
   - *Время: 5 минут*

2. **[Ваша первая пакетная обработка](first-batch.md)**
   - Обработайте несколько переводов
   - Работайте с CSV файлами
   - Создавайте отчеты
   - *Время: 10 минут*

3. **[Перевод с автокоррекцией](translation-with-qa.md)**
   - Создавайте ИИ-переводы
   - Автоматическая проверка качества
   - Итеративное улучшение (цикл TEaR)
   - *Время: 10 минут*

## Траектория обучения

```
Быстрый старт (5 мин)
    ↓
Первая пакетная обработка (10 мин)
    ↓
Перевод с автокоррекцией (10 мин)
    ↓
Готовы изучать инструкции! →
```
