# ट्यूटोरियल

**सीखने-उन्मुख** शुरुआती लोगों के लिए चरण-दर-चरण मार्गदर्शिकाएँ।

ट्यूटोरियल आपको KTTC को करके सीखने में मदद करने के लिए डिज़ाइन किए गए हैं। प्रत्येक ट्यूटोरियल एक पूर्ण पाठ है जो आपको शुरुआत से अंत तक ले जाता है।

## उपलब्ध ट्यूटोरियल

1. **[त्वरित शुरुआत गाइड](quickstart.md)** ⭐ *यहाँ से शुरू करें!*
   - KTTC स्थापित करें
   - अपनी पहली गुणवत्ता जांच चलाएं
   - MQM स्कोर को समझें
   - *समय: 5 मिनट*

## सीखने का मार्ग

```
त्वरित शुरुआत (5 मिनट)
    ↓
गाइड का अन्वेषण करने के लिए तैयार हैं! →
```
