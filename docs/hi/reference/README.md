# संदर्भ दस्तावेज़ीकरण

**सूचना-उन्मुख** तकनीकी संदर्भ और विनिर्देश।

संदर्भ दस्तावेज़ीकरण सटीक, पूर्ण तकनीकी विवरण प्रदान करता है। इस अनुभाग का उपयोग तब करें जब आपको विशिष्ट आदेश, API विधियाँ या कॉन्फ़िगरेशन विकल्प देखने की आवश्यकता हो।
